# Disk Usage Analyzer

A Python script that analyzes disk usage of files in a folder.

## Description
This project scans a directory and displays total file size along with the largest files.

## Features
- Calculates total size
- Lists largest files
- Useful for storage cleanup
- Simple and fast

## Technologies Used
- Python
- os module

## How to Run
1. Clone the repository
2. Navigate to the project folder
3. Run:

   python disk_usage.py

4. Enter folder path

## Example

Output:
Total Size: 12.45 MB  

Largest Files:
video.mp4 - 10240 KB  
image.png - 512 KB
